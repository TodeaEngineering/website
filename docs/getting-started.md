# Getting Started

## Prerequisites

- [Node.js](https://nodejs.org/) v18 or later
- npm (included with Node.js)

## Installation

```bash
git clone <repo-url>
cd website-2
npm install
```

## Development

Start the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. The site redirects to the default locale (`/en`).

Hot reload is enabled — edits to components, styles, and translations reflect immediately.

## Build & Preview

```bash
npm run build
npm run start
```

This creates an optimized production build and serves it on port 3000.

## Linting

```bash
npm run lint
```
